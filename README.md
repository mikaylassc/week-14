## Project Overview
This project is a Python application that retrieves the latest news headlines from a public REST API and displays each article’s headline and publication date. It demonstrates how to work with APIs, parse JSON data, and manage sensitive information using environment variables.

---

## API Used
This project uses the following API:

- NewsAPI.org  
- Endpoint: https://newsapi.org/v2/top-headlines  

---

## Features
- Fetches real-time news data from an external API  
- Parses JSON responses  
- Displays at least 10 news headlines  
- Shows publication date for each article  
- Includes error handling for failed requests and missing data  
- Uses environment variables for secure API key storage
